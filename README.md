# Gaokao

#### 介绍
由独立开发者“一只水晶兰”倾心打造的考试倒计时软件，坚持无广告、无收费的纯净设计理念，专为全国考生提供高效的备考辅助；它支持涵盖多种选科模式的地区识别及自定义目标，拥有可拖拽置顶的桌面悬浮窗，并创新性地根据剩余时间的紧迫程度呈现从青色到红色的动态视觉反馈，同时配备基于日期的验证码安全退出机制，全方位助力学子把握备考节奏，金榜题名。

#### 安装使用
点击此处链接即可下载，你需要确保已安装指定版本的Python，目前支持Python 3.8到Python 3.12的所有版本，所有代码文件放在同一目录，直接运行main文件就能启动。

https://gitee.com/PythonSJL/gaokao/releases/tag/v1.0.1
